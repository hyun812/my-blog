## 💡 주요 기능

- MDX를 활용하여 마크다운 기반 블로그 포스팅 작성 환경 구축

- 반응형 디자인을 적용하여 다양한 디바이스에서 일관된 사용자 경험 제공

- 다크/라이트 모드 지원

- Giscus를 이용한 댓글 기능 구현

- TOC (Table of Contents) 구현

- 사이트맵 생성 및 구글 검색 엔진 등록

- Google Analytics 연동
